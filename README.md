# AUTO REPUBLIC MANAGER

Architecture de l'application de gestion automobile professionnelle.

## Objectif

Une plateforme web moderne pour gérer un showroom et un garage dans une seule solution.

## Structure

- `frontend/` : application Next.js + TypeScript + Tailwind CSS
- `backend/` : API AdonisJS + Lucid ORM + JWT
- `database/` : schéma MySQL et data-models initiaux

## Priorités créées

1. architecture complète
2. dashboard principal
3. sidebar professionnelle
4. routes frontend/backend
5. schéma MySQL
6. premières pages UI HD

## Instructions de démarrage

- Frontend : `cd frontend && npm install && npm run dev`
- Backend : `cd backend && npm install && npm run dev`
